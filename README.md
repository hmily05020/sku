## 功能

- 现售机制：展示各部门当前售卖 SKU、售价，并按维护的单品成本估算毛利率和保本 ROI。
- 现售机制编辑：可新增、编辑、删除 SKU；改动保存在浏览器本地。
- 渠道口径：现售页和快速测算都支持自营 / 达人带货，自营不计达人佣金和团长佣金。
- 单品退货率：编辑现售 SKU 时可单独填写退货率；留空则使用现售页全局退货率。
- 快速测算：输入机制、售价、退货率、平台扣点、电商平摊、佣金等参数，实时计算保本 ROI。
- 成本维护：按“产品 + 组件”维护成本，例如 `98g咖啡糖 + 98g = 8`。`通用` 产品成本只作为兜底。

## 计算口径

- 发货成本 = 产品成本 + 快递&打包 + 平台扣点 + 电商平摊 + 人工&其他 + 运费险
- 毛利 = 售价 - 发货成本 - 达人佣金 - 团长佣金
- 去退保本 ROI = 售价 / 毛利
- 含退货保本 ROI = 去退保本 ROI / (1 - 退货率 + 退货修正率)

当前默认退货修正率为 `1%`，用于匹配旧测算表里的 `=$S3/(1-10%+1%)` 口径。
